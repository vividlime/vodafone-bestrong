This development environment requires the following to be installed:


1. Node (NPM)
https://nodejs.org/


2. Ruby
https://www.ruby-lang.org/en/


3. Compass 0.12.7
gem install compass -v 1.0.3


4. Bower
npm install bower -g


5. Grunt
npm install -g grunt-cli



Install all dependencies :

npm install
bower install
